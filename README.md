## 📌 Overview
This project analyzes coffee shop sales data using **SQL** and **Power BI** to uncover key business insights.  
The goal is to transform raw transactional data into meaningful, data-driven decisions.

---

## 🎯 Objectives
- 📊 Analyze total sales, orders, and quantity sold  
- 📈 Perform Month-over-Month (MoM) growth analysis  
- ⏰ Identify peak hours and busiest days  
- 🛍️ Analyze product category performance  
- 📍 Compare store locations  
- 🔝 Identify top 10 best-selling products  

---

## 🗂️ Dataset
The dataset includes:
- Transaction Date & Time  
- Product Category & Type  
- Unit Price & Quantity  
- Store Location  

---

## ⚙️ Tools & Technologies
- 🛢️ PostgreSQL (SQL)
- 📊 Power BI
- 🧠 DAX
- 📁 CSV / Excel

---

## 📊 Key Analysis

### 🔹 Sales Analysis
- Monthly sales trends  
- MoM growth using SQL window functions (`LAG`)

### 🔹 Time-Based Insights
- Peak hours → `EXTRACT(HOUR)`  
- Peak days → `EXTRACT(DOW)`  

### 🔹 Business Insights
- Weekday vs Weekend comparison  
- Store-wise performance  
- Product category contribution
